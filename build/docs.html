Clarity Scan Documentation

About Clarity Scan

Clarity Scan is a Chrome Extension designed to analyze YouTube content and detect "Brain Rot" content by processing video transcripts.

What is a Brain-Rot YouTube Video?

A brain-rot YouTube video is one that prioritizes sensationalism, misinformation, or mindless entertainment over substance, critical thinking, or meaningful value.

Clarity Scan helps detect such content, allowing users to make informed decisions about the media they consume.

Brain-Rot Scoring System

Clarity Scan assigns a Brain-Rot Score ranging from 0% to 100%, in increments of 10%:

  • 0%: No brain-rot, high-quality and informative content.
  • 10%-40%: Slightly low in substance but still informative.
  • 50%-70%: Noticeable use of clickbait or misleading information.
  • 80%-100%: Primarily focused on sensationalism, misinformation, or low-value entertainment.

The score is calculated using AI-driven content analysis alongside our custom Python model.

How We Grade Brain-Rot:

  • We first process the video transcript and generate a summary.
  • The summary is analyzed by an AI model trained to detect patterns of misinformation, clickbait, or sensationalism.
  • We then send this AI-generated review into our custom Python model, which assigns a final Brain-Rot Score based on our internal parameters.
  • This approach ensures a balanced evaluation, reducing bias while improving accuracy.

Features

  • Analyzes YouTube videos in real-time.
  • Detects low-quality or misleading content.
  • Uses both AI analysis and a Python-based scoring model.
  • Provides an intuitive UI for reviewing results.

Permissions & Security

Clarity Scan requires certain permissions to function:

  • Storage: To store user preferences.
  • ActiveTab: To analyze content in the current tab.
  • Notifications: To alert users about detected content.

How to Use

  1. Install Clarity Scan from Github.
  2. Open any YouTube video.
  3. Click on the Clarity Scan icon in the extension bar.
  4. Review the analysis results in the popup.
